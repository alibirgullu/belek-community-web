# Belek University Community Platform – Web Application

This repository contains the web-based frontend of the Belek University Student Communities Platform.

## Technologies
- React
- REST API integration
- Responsive Design (Mobile-first)

## User Interfaces
- Student Interface
- Community Management Panel
- SKS (Super Admin) Panel

## Role-Based Access
All panels are implemented within the same web application.
Different user roles access different screens based on authorization.

## Features
- Community discovery and membership
- Event and announcement management
- Admin dashboards
- Notification system

## Backend Integration
This application consumes the REST APIs provided by:
- belek-community-backend
